
# Short Questions Answers

1. Docker image vs container: Image is template; container is running instance.
2. start vs enable: start runs now; enable runs at boot.
3. Nginx reverse proxy: routes traffic to backend services.
4. Check port usage: sudo lsof -i :PORT
5. AWS EC2: cloud virtual servers.
6. Jenkins: CI/CD automation tool.
7. CodePipeline: AWS managed build/test/deploy pipeline service.
